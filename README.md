# Fuzzy Name Conversion Tool

A tool that performs fuzzy string matching and name conversion using string similarity algorithms. Useful for matching names with typos, abbreviations, or variations.

## ✨ Features

- Fuzzy matching of similar names
- Use of Levenshtein Distance and other algorithms
- Handling spelling variations and errors
- Match scoring output

## 🔧 Tech Stack

- Python 3
- FuzzyWuzzy / RapidFuzz
- Pandas
